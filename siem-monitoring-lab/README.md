# SIEM-Based Security Monitoring Home Lab

## Objective
To build a controlled lab environment to understand how SIEM platforms collect logs, generate alerts, and support incident analysis.

## Lab Environment
- Host Operating System: Windows
- SIEM Platform: Splunk Enterprise
- Activity Generator: Kali Linux (Virtual Machine)

## Scope of Work
- Configure Splunk to ingest Windows security and system logs
- Generate controlled authentication-related security events
- Analyze log data to identify suspicious activity patterns
- Document observations and incident analysis

## Key Learnings
- Understanding SIEM log ingestion and indexing
- Familiarity with Windows authentication event IDs
- Practical exposure to SOC-style investigation workflows
